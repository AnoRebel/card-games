# Design Context — Card Games Platform

## Target audience
Casual players (mobile-first) playing Last Card and Albastini with friends —
offline (hotseat/bots) and online (realtime rooms). All ages, global (en + sw).

## Use cases
Pick a game → create/join a room or play offline → play hands of cards → chat,
see leaderboard, learn via tutorial/rules. Sessions are short and social.

## Brand personality / tone
**Gamey, tactile, alive — never boring, never "AI slop".** It should feel like
sitting at a real table to play, with satisfying motion and feedback. Clean,
confident, mobile-friendly, fast. Two games each keep a subtle personality
(Last Card = playful/modern via Outfit; Albastini = heritage via Fraunces).

## Selectable visual themes (app-wide, switchable in settings)
1. **Felt** (default) — premium casino: green/charcoal felt, warm gold accents,
   tactile cards with depth/shadow. Classy through tactility.
2. **Arcade** — vibrant playful: bold saturated colors, chunky rounded shapes,
   springy energy. Casual mobile-game feel.
3. **Neon** — dark luxe: dark surfaces, one electric accent, glowing card edges,
   sharp snappy motion.

## Motion
- A **motion-intensity preference** in settings: **Rich** (default — tasteful +
  flashy combined: staggered deals, flips, slides to winner, win celebration,
  hover tilt, press feedback) vs **Subtle** (fast, minimal). Anime.js drives
  event sequences. ALWAYS honor prefers-reduced-motion (snap to final).

## Layout / spacing
- **Tighter, uniform spacing** by default (small, consistent) unless a design
  standard calls for breathing room. Avoid the large/small inconsistency.
- Mobile-first and genuinely responsive — adapt, don't amputate.
- **Minimal navigation**: clear, few destinations. Settings is a **global
  slideover/modal**, not a separate page. No redundant chrome.

## Extra
- Move feedback: surface a short description of each move (accordion / log).
- Notifications should feel lively, not plain toasts.

## Constraints
Nuxt 4 + @nuxt/ui (Tailwind v4), @vueuse/motion + anime.js, reduced-motion safe.
Fonts: Bricolage Grotesque (display), Inter (body), Outfit (Last Card), Fraunces
(Albastini). No pure black/white; tint neutrals toward the active theme hue.
